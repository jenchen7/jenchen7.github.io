Spiritwood

Spiritwood

Play as a hunter attempting to revitalize their town after it was abandoned. Engage with the tight combat system against various enemies and improve combat abilities for better survival in this top-down roguelite action game.

Role: Developer
Team size: 3
Developed: September 2024-February 2025
Tools: Unity2D, Aseprite, Audacity, Git.
Link: https://dragonstar7.itch.io/spiritwood

Features:

  • Tight combat system allows for multiple options the player can use against different enemies.
  • Semi-randomized dungeon and random enemy spawns makes each run a different experience.
  • Intuitive loot and upgrade system to empower the player.

Role Overview:

  • Created 6 unique enemy types using state machines. Enemies have their own mechanics and behaviour that require the player to test what works best against them.
  • Created the loot system and upgrade loop and implemented the UI to support it. This includes the player's inventory, enemy loot drops, and the upgrade shop. The upgrade shop has 6 different improvement options for the player's offensive and defensive capabilities.
  • Implemented the death and respawn system, for a faithful roguelite experience.
  • Implemented the audio system and other minor additions to the game.

Smart Campus: Virtual Campus
A university project creating an interactive map navigation system for the campus.

Smart Campus: Virtual Campus

The Smart Campus: Virtual Campus project is to create an interactive map and navigational system for the university campus, featuring a faitful and accurate recreation to scale. It aims to reduce the confusion students and other visitors may have when traversing the campus for the first few times.

Role: Programmer, Modeller
Developed: May 2023-December 2023
Tools: Unity3D, Autodesk Maya, Git

Features:

  • Utilizes the Unity AI Navigation system to path the shortest route possible among the main campus paths.
  • There are two options for campus navigation: to instantly traverse to any building of the main campus, and to set an auto navigation path from the current point to a selected building. This can be selected in the menu.
  • Manual navigation is possible for a slower change of pace.

Role Overview: I modelled around half of the buildings using Autodesk Maya. I also did all the Unity programming, covering all the mentioned features.

Jump Guy
A colourful take on the Google Chrome dinosaur game.

Jump Guy

Jump Guy, an infinite 2D side-scroller/platformer, is a colourful take on the Google Chrome dinosaur game. It tests how long you can last using not only time, but score. With the ever-increasing speed and the possibility scarcity of points, this simple game can provide widely different results every time.

Role: Solo developer
Developed: May 2024-July 2024
Tools: Unity3D

Features:

  • Randomized generation of objects, from a pool of enemies, collectables, and obstacles.
  • Infinite gameplay potential with ever-increasing player speed.
  • Player has 3 lives, ending the game when all are depleted.
  • Clean UI to track player lives, score, and time.
  • Object cleanup system for offscreen objects.
  • Player and menu animations.
